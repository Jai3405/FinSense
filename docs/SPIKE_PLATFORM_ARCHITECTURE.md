## EXECUTIVE VISION

It serves:
- The **beginner** who doesn't know what to buy
- The **intermediate** investor who wants smarter decisions
- The **active trader** who needs real-time intelligence
- The **long-term investor** who wants set-and-forget optimization
- The **HNI** who needs institutional-grade analytics
- The **enterprise** that wants to embed intelligence into their platform

## USER JOURNEYS

### Journey 1: The Beginner (₹5K-50K portfolio)
**Pain Points:**
- "I don't know what to buy"
- "Stock market seems scary"
- "I don't understand financial jargon"

**SPIKE Solution:**
1. **Onboarding Quiz** → Determine risk profile, goals, horizon
2. **Smart Themes** → "Start with India Growth Basket"
3. **FinScore** → Simple 0-10 rating they understand
4. **Guided Investing** → "Invest ₹5000? Here's how to split it"
5. **Learn Mode** → Every decision explained simply

---

### Journey 2: The DIY Investor (₹50K-5L portfolio)
**Pain Points:**
- "I pick stocks but don't know if they're good"
- "My portfolio is all over the place"
- "I miss important news about my holdings"

**SPIKE Solution:**
1. **Portfolio Import** → Sync from broker or manual entry
2. **Portfolio Health Score** → "Your portfolio scores 62/100"
3. **FinScore for Holdings** → See which stocks are weak
4. **Legend Analysis** → "Buffett would say..."
5. **Smart Alerts** → "HDFC Bank sentiment dropping"
6. **Rebalancing Suggestions** → "Consider trimming X, adding Y"

---

### Journey 3: The Active Trader (Intraday/Swing)
**Pain Points:**
- "I need real-time signals"
- "I overtrade and lose money"
- "I can't track everything"

**SPIKE Solution:**
1. **Trading Dashboard** → Real-time charts, signals
2. **AI Confidence Meter** → "BUY confidence: 73%"
3. **Regime Indicator** → "Market in high volatility mode"
4. **Execution Intelligence** → "Best time to enter: 11:30 AM"
5. **Behavioral Guard** → "You've traded 5x today. Slow down?"
6. **Paper Trading** → Practice without risk

---

### Journey 4: The Long-Term Investor (5+ year horizon)
**Pain Points:**
- "I want to invest and forget"
- "I don't want to check daily"
- "I want steady compounding"

**SPIKE Solution:**
1. **Portfolio Autopilot** → Fully automated management
2. **Quarterly Rebalancing** → Automatic, tax-efficient
3. **Goal Tracking** → "You're 67% to your ₹1Cr goal"
4. **Quality Focus** → Only high-FinScore compounders
5. **Dividend Tracking** → Reinvestment automation
6. **Monthly Report** → Simple email summary

---

### Journey 5: The HNI (₹50L+ portfolio)
**Pain Points:**
- "I need institutional-grade analysis"
- "I want personalized strategies"
- "Tax optimization is critical"

**SPIKE Solution:**
1. **Advanced Analytics** → Factor exposure, VaR, stress tests

### Technology Stack

| Layer | Technology | Rationale |
|-------|------------|-----------|
| **Web Frontend** | Next.js 14 + TypeScript | SSR, great DX, scalable |
| **Mobile** | React Native | Cross-platform, shared logic |
| **API Gateway** | Kong / AWS API Gateway | Rate limiting, auth |
| **Backend Services** | FastAPI (Python) | ML integration, async |
| **Real-time** | WebSocket + Redis Pub/Sub | Live updates |
| **Database** | PostgreSQL + TimescaleDB | Relational + time-series |
| **Cache** | Redis | Session, rate limiting |
| **Search** | Elasticsearch | Stock search, filters |
| **ML Serving** | FastAPI + PyTorch | Model inference |
| **LLM** | Claude API / GPT-4 | Strategy-GPT, explanations |
| **Message Queue** | RabbitMQ / Kafka | Async processing |
| **Storage** | S3 / CloudFlare R2 | Files, reports |
| **CDN** | CloudFlare | Static assets |
| **Monitoring** | Prometheus + Grafana | Observability |
| **Logging** | ELK Stack | Centralized logs |
